## Summary File Format

Summaries should be placed in `/summaries/` directory with this naming:

```
summaries/
  01-Introducing-AI-Driven-Development_01-moment_that_changed_everything.md
  01-Introducing-AI-Driven-Development_02-three-trillion-developer-economy.md
  ...
```

### Summary File Structure

```markdown
---
original_path: 01-Introducing-AI-Driven-Development/01-ai-development-revolution/01-moment_that_changed_everything.md
chapter: 01-Introducing-AI-Driven-Development

(200-400 words)

- AI-driven development fundamentals
- The paradigm shift in coding
- Developer productivity transformation
- Economic impact on software industry
- Future opportunities for developers
```

## States

### 1. Loading
Shows spinner while fetching summary

### 2. Summary Available
Displays:
- Difficulty badge (beginner/intermediate/advanced)
- Read time badge
- Summary text (200-400 words)
- Key concepts list (3-5 items)
- "Generated by OLIVIA AI Tutor" footer

### 3. Summary Not Available
Shows:
- üîß Icon with bounce animation
- "Team Working on it!" message
- Purple gradient background
- "Check back soon!" text
